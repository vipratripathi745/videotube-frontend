# 🎬 VideoTube

A full-stack YouTube-inspired video streaming platform built using the MERN Stack.

---

## 🚀 Live Demo

🌐 Frontend: https://videotube-frontend-cyan.vercel.app

⚙️ Backend: https://video-backend-9rxp.onrender.com

---

## ✨ Features

- 🔐 JWT Authentication
- 👤 User Registration & Login
- 📤 Upload Videos
- 🎥 Watch Videos
- ❤️ Like / Unlike Videos
- 💬 Comment System
- 🔔 Subscribe / Unsubscribe
- 📺 User Channels
- 📂 Playlists
- ⏰ Watch Later
- 🕒 Watch History
- 🔍 Search Videos
- 📊 Creator Dashboard
- ✏️ Edit Video Details
- 🗑 Delete Videos
- 🌍 Fully Responsive
- ☁️ Cloudinary Media Storage

---

### Frontend

## ⚙️ Installation

### Clone

```bash
git clone https://github.com/vipratripathi745/videotube-frontend
```

### Frontend

```bash
cd videotube-frontend
npm install
npm run dev
```

### Backend

```bash
cd video-backend
npm install
npm run dev
```
